## Descrição

Construa um formulário de endereço que preencha automaticamente o endereço por meio do CEP, conforme ilustrado na _Figura 1_.

_Figura 1: Preenchimento automático do endereço por meio do CEP._

<Image
  src={layout}
  alt="Preenchimento automático do endereço por meio do CEP"
  class="border p-4"
/>

A consulta de informações de endereço por meio do CEP deve ser feita utilizando a API do [Via CEP](https://viacep.com.br), para realizar a consulta, basta fazer uma requisição HTTP para a URL `https://viacep.com.br/ws/{CEP}/json/`, no qual o `{CEP}` é o CEP que deseja consultar. Por exemplo, para consultar o CEP `01001-000`, basta fazer uma requisição HTTP para a URL [`https://viacep.com.br/ws/01001000/json/`](https://viacep.com.br/ws/01001000/json/), e resultao será:

```json
{
  "cep": "01001-000",
  "logradouro": "Praça da Sé",
  "complemento": "lado ímpar",
  "bairro": "Sé",
  "localidade": "São Paulo",
  "uf": "SP",
  "ibge": "3550308",
  "gia": "1004",
  "ddd": "11",
  "siafi": "7107"
}
```
