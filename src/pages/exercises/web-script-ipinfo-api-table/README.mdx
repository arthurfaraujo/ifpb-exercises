## Descrição

Crie um aplicativo que exiba as informações do IP do usuário em uma tabela que tenha interface parecido com a da _Figura 1_.

_Figura 1: Busca de Informações do IP._

<Image src={layout} alt="Busca de Informações do IP" class="border p-4" />

Para obter informações do IP, utilize a API [Ipinfo](https://ipinfo.io/developers). Por exemplo, para obter informações do IP `8.8.8.8`, utilize a URL `https://ipinfo.io/8.8.8.8/json?token=TOKEN`.
