## Descrição

O comando `ping` é um utilitário de rede que permite verificar se um dispositivo está conectado à rede. O comando `ping` envia pacotes de dados para um dispositivo e aguarda a resposta. Se o dispositivo responder, o comando `ping` exibe uma mensagem informando que o dispositivo está conectado à rede. Caso contrário, o comando `ping` exibe uma mensagem informando que o dispositivo não está conectado à rede.

Para auxiliar a utilização do comando `ping`, crie uma página Web que descreva os detalhes e parâmetros do comando. A página deve ser criada com HTML e deve seguir o layout apresentado na _Figura 1_.
